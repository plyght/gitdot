# Getting started

gitdot is a GitHub alternative for open-source maintainers. This guide walks you
through creating an account, pushing your first repository, and inviting collaborators.

## Create an account

Head to the signup page and create an account with your email. We'll send you a
one-time code to verify it.

## Push your first repo

Create a repository from the dashboard, then push your local code:

```bash
git remote add origin https://gitdot.dev/your-name/your-repo.git
git push -u origin main
```

That's it — your repository is now live.
