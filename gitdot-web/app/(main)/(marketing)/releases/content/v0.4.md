# v0.4 — Reviews, finally.

Code review lands in gitdot. Open a pull request, leave line-level comments, and
approve changes — all with the same fast, incremental loads you get everywhere else.

## What's coming

- Pull requests with side-by-side and unified diffs.
- Line-level and file-level review comments.
- Approvals and change requests that gate merges.

This is the release we've wanted to ship since day one. More details as it nears.
