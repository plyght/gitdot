gitdot's very first public release. \
We recognize that things may be shaky still, but wanted to share what we've been up to.

**1. Users**
You can signup! (and customize your profile as you'd like).
The two authentication modes we support are: passwordless email authentication and GitHub OAuth.

**2. Orgs**
You can create an org and add members too.
Orgs also have a profile and can host repositories too (private and public, just like GitHub).

**3. Repositories**
You can create, clone, and push repositories via HTTP.
To authentiate pushes and private clones, you'll have to install the gitdot-cli.

**4. GitHub backups:**  
You can import repositories from GitHub into gitdot.
By default, we import them as read-only mirrors and automatically sync commits from GitHub to gitdot.
At any time, if you'd like to migrate over, you can do so by promoting the repo in settings

**5. A very fast website:** 
You can click around, star, and see what the rest of gitdot has been up to in a very-fast, CLI-inspired website.
