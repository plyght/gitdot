# v0.1 — First public preview

The first public preview of gitdot is here. This release lays the foundation for
everything to come.

## Highlights

- A hyper-performant Git server written in Rust, speaking the smart HTTP protocol.
- Repository browsing on the web with fast, incremental loads.
- Email one-time-code and GitHub OAuth sign-in.

## Known limitations

This is a preview — expect rough edges. CI/CD and the issue tracker are still in
development and will land in upcoming releases.
